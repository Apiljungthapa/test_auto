# Invoice Pipeline Setup

This is an automated invoice processing pipeline application.

## Project Structure

- `InvoicePipeline/` - Main application code
  - `app.py` - Flask application entry point
  - `api_client.py` - API client for processing
  - `config.py` - Configuration settings
  - `main.py` - Main processing logic
  - `excel/` - Excel file handling
  - `reports/` - Report generation modules

## Requirements

See `requirements.txt` for dependencies.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python InvoicePipeline/app.py
```
